현재 폴더 : 작업 중인 폴더 ==> . 상위 폴더(부모 폴더) ==> .. 하위 폴더(자식 폴더) ==> 폴더명 최상위 폴더(루트 폴더) ==> / ========================== 경로 : 특정한 파일이 있는 위치를 나타내는 표식 1. 절대 경로 : 루트를 기준으로 찾아가는 경로 ( '/'로 시작한다. 2. 상대 경로 : 현재 폴더를 기준으로 찾아가는 경로 ('./'로 시작한다. './'는 생략 가능) =========================== * 경로에 사용되는 '/'문자의 역할 ==> 경로의 중간에 있는 '/'문자 ==> 폴더와 폴더, 또는 폴더와 파일을 구분하는 구분자 역할을 한다. ===> 경로의 첫번째 위치의 '/'문자 ==> 루트를 나타낸다.
